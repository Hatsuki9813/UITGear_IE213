## 📖 Cách sử dụng 
### 1. Yêu cầu hệ thống
- Node.js (phiên bản mới nhất)
- MongoDB
- Git
- Thiết bị di động hoặc trình giả lập phù hợp
### 2. Cách cài đặt
1. Clone repository:
   ```bash
   git clone https://github.com/Hatsuki9813/UITGear_IE213.git
   ```
2. Cài đặt các thư viện cần thiết:
   ```bash
   cd frontend
   npm install
   cd backend
   npm install
   cd admin
   npm install
   ```
3. Chạy dự án:
   ```bash
   cd backend
   npm run dev
   cd frontend
   npm run dev
   cd admin
   npm run dev
   ```
4. Mở dự án trên trình duyệt
